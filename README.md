# Portfólio Pessoal - Haslan Salgado

## Descrição do Projeto
Este projeto consiste em uma página web de portfólio pessoal desenvolvida durante os estudos de desenvolvimento Front-end.

A página apresenta informações pessoais, habilidades técnicas e comportamentais, projetos desenvolvidos e um formulário de contato.

## Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript
- Font Awesome (ícones)

## Estrutura do Projeto

index.html – Estrutura principal da página  
style.css – Estilização da página  
script.js – Funcionalidades em JavaScript  
image-atual.jpeg – Foto do perfil  
README.md – Documentação do projeto
